The following libraries are used by SQuirreL:

  • Apache Commons CLI 1.0 (http://jakarta.apache.org/commons/cli/) Command Line Interface.
  • Apache log4j (http://jakarta.apache.org/log4j) Logging API.
  • jGoodies Forms 1.0 (https://forms.dev.java.net/) Forms Layout System.
  • NanoXML 2.1 (http://web.wanadoo.be/cyberelf/nanoxml/) Java XML API.
