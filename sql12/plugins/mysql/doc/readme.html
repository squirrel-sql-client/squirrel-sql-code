Introduction

The MySQL plugin adds MySQL specific functionality to SQuirreL SQL Client.

New Tabs

The session node in the object tree has two new tabs added. The "Status" tab display status information about the MySQL server.

And the "Processes" tab shows the current processes attached to the server.

To display the information in these tabs successfully you need to have sufficient authority to run the show status and show full processlist commands against the server.

The Catalog nodes in the object tree have a new "Open Tables" tab. To display this tab successfully you need to have the rights to run show open tables against the catalog.

New Menu Options

A new option "MySQL Create table Script" has been added to the popup menu for tables in the Object Tree. This will generate a "Create Table" script for the selected table(s) and add it into the SQL entry area.
